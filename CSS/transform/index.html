1. translate(x-axis, y-axis): it moves an element from it's current position wrt the X and Y Axis values given.
  2. rotate(deg): it rotates your element counter clockwise.
  3. scale(x-axis, y-axis): it increases and decreases the size of the element
  4. skew(deg): it skews an element along the x and y axis
  5. matrix(scaleX, skewY, skewX, scaleY, translateX, translateY ): combines all the transform methods into one.
