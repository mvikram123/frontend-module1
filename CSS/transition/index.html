CSS Transitions: allow you to change property values smoothly over a given duration.

  1. transition: shorthand property
  2. transition-delay: it species when will the transition effect start.
  3. transition-duration: it specifies that for how much time the effect should last
  4. transition-property: it specifies the css property to which you want to apply the transition to.
  5. transition-timing-function: specifies the speed curve of the transition effect
    • ease, linear, ease-in, ease-out, ease-in-out, cubic-bezier(n,n,n,n)
